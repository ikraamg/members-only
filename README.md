# Members Only

> In this project, we implemented the authentication system using the Devise Gem so users can only access areas of a site they are authorized to. Only signed-in users can write posts and can see who the author of a post is.

## Live Demo

[Live Demo Link](https://obscure-oasis-88086.herokuapp.com/)

## Built With

- Ruby 2.6.5 and Rails 5.2.4.2
- Bootsrap 4
- Deployed on Heroku

## Getting Started

To get a local copy up and running follow these steps:

### Prerequisites

- Ruby installed on local machine

### Usage

- Fork/Clone this project to your local machine
- Open folder in your local enviroment and run thes lines of code to get started:

```Ruby
cd members-only
bundle install
rails db:migrate
rails s
```

- Open a web page and go to [port 3000 on your local machine.](http://localhost:3000)
- The original project can be found on this [Odin Project page](https://www.theodinproject.com/courses/ruby-on-rails/lessons/authentication?ref=lnav).
